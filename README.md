# Bienvenido a Frontend Perseo


 El Frontend  de la **prueba técnica** is creado con cariño & usa tecnologías como:
 - ReactJS.
 - Hooks.
 - Axios.
 - CryptoJS.
 - AntDesign.
 - QS.
 - JSONWebToken.
 - Body Parser.

### Instalar dependencias y ejecutar:
>yarn install

>yarn start
